# Callback C002 — pch.h 첫 수정 포인트 교정

P1 §1(pch.h) 방향 교정:

- pch.h 재정렬.
- crtdbg 블록을 pch.h에서 **framework.h 하위로 이동** → Windows 경계 코드로 전환.
- framework.h Linux 분기에 **Linux 전용 crtdbg 경계(대응 placeholder)** 추가.

→ pch.h는 인라인 플랫폼 가드를 갖지 않고, 플랫폼 분기는 framework.h가 단독 소유.
