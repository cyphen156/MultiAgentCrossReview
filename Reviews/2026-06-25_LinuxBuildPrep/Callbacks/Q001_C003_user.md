# Callback C003 — 디버그 출력 shim을 플랫폼 매크로로 전환

- inline 함수 대신 **플랫폼 디버그 출력 매크로**로 전환.

- 이러면 기존 코드를 그대로 살릴 수 있고, 호출부는 함수명만 매크로로 살짝 바꾸면 됨.
